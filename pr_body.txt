- `db.js` 생성: `mysql2/promise`의 `createPool`을 사용해 DB 커넥션 풀 구성
- `routes/products.js`: `GET /api/products`에서 더미 데이터(`dummyProducts`) 제거 후 실제 DB 조회(`SELECT * FROM products`)로 교체
- 비동기 처리(`async/await`) 적용 및 `try/catch`로 에러 핸들링 추가 (500 응답 + 서버 로그)
- `db-test.js`: 하드코딩된 접속 정보를 `.env` 기반 `process.env`로 변경
- `.env`에 `DB_HOST`, `DB_PORT`, `DB_USER`, `DB_PASSWORD`, `DB_NAME` 추가 (`.gitignore`에 포함되어 있음 확인 완료)

## 테스트
- `db-test.js`로 DB 연결 및 `SHOW TABLES` 조회 성공 확인
- `GET /api/products` 호출 시 실제 DB 조회 결과 반환 확인
